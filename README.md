## Nezuko



## Install 
```bash

$ npm install && npm update
# Install Google Chrome and FFmpeg, change the config.json file and Credentials.
$ npm start

```
## Usage
```js
/*
        「 𝐅𝐮𝐧 ┆ ⛓ 」
        ╔══════════════╗
    
        !say: I say what you want
        !kill: Kill someone in a group
        !joke: Prank someone in the group.
        !slap: Slap someone in a group.
        !kiss: Kiss someone in a group.
        !cry: Ponerte a Llorar.
        !sleep: Make you cry.
        !8ball: Typical 8ball game.
        !stupidmeter: Measure your level of stupidity and that of your friends!
        ╚══════════════╝
        
        
                    「 *Info* ┆ 📃 」
        ╔══════════════╗
    
        !chats: Number Of Chats Currently Open.
        ╚══════════════╝
        
        
                「 *Groups* ┆ 👩🏼‍🤝‍🧑🏻 」
        ╔══════════════╗
        !groupinfo: Group information.
        !desc: Change the description of the group if you have the respective permissions.
        !title: Change the Title of the group if you have the respective permissions.
        
        ╚══════════════╝  
        
        
                    「 *Util* ┆ 🔧  」
                    
                    
        ╔══════════════╗
        
        !sticker: Turn a photo into a sticker.
        !youtube: Copy and paste a Link of a Video next to the command to download it.
        !youtubemp3: Copy and paste a Link of a Video together with the command to download the audio.
        !ip: Displays the Geographical Data of the Desired IP.
        !about: See someone's status in a group.
        !math: Solve simple mathematical exercises.
        !clima: Indicates the Climate of a City.
        !avatar: View another person's Avatar / Profile Picture in a group.
        
        ╚══════════════╝
        
        
                「 *Crypto* ┆ 💳 」
                
                
        ╔══════════════╗
    
        !bitcoin:  Displays the Current Price of Bitcoin.
        !ethereum: Shows the Current Price of Ethereum.
        !litecoin: Shows the Current Price of Litecoin.
    
        ╚══════════════╝

*/
```
